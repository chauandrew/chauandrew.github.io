Sharded Key-Value Store

Sharded Key-Value Store

A distributed key-value server featuring a fault-tolerant configuration service and replicated server groups. The configuration service receives requests to join and leave from replicated server groups and maintains a consistent configuration to all groups. The configuration service and server groups utilize a self-written Paxos library to maintain consistency.

Technologies Used: Go

Taxi Ride Duration Model

Taxi Ride Duration Model:

A machine learning model trained on over 80,000 data points to predict taxi ride durations in Manhattan. Utilizes complex feature selection and engineering, such as using PCA to separate the city into Upper, Lower, and Midtown Manhattan. Cross validation was applied across several SVM kernels and hyperparameters, ultimately resulting in an SVM with a Gaussian RBF kernel

Technologies Used: Python3, Scikit-Learn, Jupyter Notebooks

TuneSearch

TuneSearch:

A search engine for songs by lyrics. Songs are ranked using term frequency-inverse document frequency (TFIDF) scores, which scores lyrics using their relative frequency in a song and their frequency in the entire database. Users can search for songs using an AND or an OR query, and search results are cached for faster pagination.

Technologies Used: PostgreSQL, Python, Flask
